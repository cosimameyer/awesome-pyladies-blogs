About This Directory

What is this?

This is a personal, independent project — not an official PyLadies initiative. It is not affiliated with or endorsed by the PyLadies organisation.

Where does this content come from?

Everything featured here is sourced from the open-source repository awesome-pyladies-creations on GitHub. Each person is reached out to individually before being added — so while this is a personally maintained project, the list is community-driven at heart. The site rebuilds automatically whenever new entries are added, and more PyLadies members are joining all the time.

Want to be added?

If you're a PyLadies member and would like your work featured, contributions are very welcome! Check out the contributing guide — it only takes one JSON file.

Don't want to be featured?

If you'd prefer your content or profile not to appear on this site, please don't hesitate to reach out. You can open an issue on GitHub or contact me directly via cosimameyer.com and I'll take care of it as soon as possible.

Are you a PyLady?

Add your blog, package, YouTube channel, or other work to the directory — it takes just one JSON file.
