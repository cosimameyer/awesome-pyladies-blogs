Packages & Tools

Python packages and software built and maintained by PyLadies members.

CiteLang

Generate credit summaries and badges for software packages to ensure proper attribution.

Maintained by Vanessa Sochat
AX

axe-playwright-python

Automated web accessibility testing using axe-core engine and Playwright.

Maintained by Pamela Fox

deid

Best-effort de-identification of DICOM medical images using Python and pydicom.

Maintained by Vanessa Sochat

emojificate

Convert emoji in HTML to accessible fallback images with alt text, title text, and aria labels.

Maintained by Katie McLaughlin

expfactory

Software to generate a reproducible container with a battery of experiments.

Maintained by Vanessa Sochat

fairlearn

A Python package to assess and improve fairness of machine learning models.

Maintained by Hilde Weerts, Adrin Jalali, Miroslav Dudík + 6 others
FL

flask-talisman

A small Flask extension that handles HTTP security headers for you.

Maintained by Thea Flowers, jonakemon, Jannis Leidel
GI

gidgethub

An async GitHub API library for Python.

Maintained by Mariatta Wijaya, Brett Cannon

ih

A Python package for creating cross-stitch embroidery patterns from images.

Maintained by Katie McLaughlin

kedro

An open-source Python framework for creating reproducible, maintainable and modular data science code.

Maintained by Merel Theisen, Yetunde Dada

nox

Flexible test automation for Python.

Maintained by Thea Flowers, Danny Hermes, Chris Wilcox + 3 others

oras

Python SDK for OCI Registry as Storage (ORAS) — push and pull arbitrary artifacts to and from OCI registries.

Maintained by Vanessa Sochat

overviewpy

Easily Extracting Information About Your Data

Maintained by Cosima Meyer

pokemon

ascii database of pokemon in Python!

Maintained by Vanessa Sochat
PU

pull-request-action

GitHub Action to open a pull request when a branch is pushed or updated.

Maintained by Vanessa Sochat
RO

roctet

Generate diverse score-target datasets with the same AUROC value, ala Anscombe's Quartet

Maintained by Emily Riederer
SI

singularity-compose

A simple orchestration library for Singularity containers, akin to Docker Compose.

Maintained by Vanessa Sochat

singularity-hpc

Local filesystem registry for containers (intended for HPC) using Lmod or Environment Modules.

Maintained by Vanessa Sochat

snakedeploy

Deployment and maintenance toolbox for Snakemake workflows.

Maintained by Vanessa Sochat, johanneskoester

spython

A Python command-line tool for working with Singularity containers.

Maintained by Vanessa Sochat

tributors

Give tribute to your repository contributors — generates contributor files like .all-contributorsrc and CITATION.cff.

Maintained by Vanessa Sochat, yarikoptic

urlchecker

A tool to collect and validate URLs across static files — useful for checking links in code and documentation.

Maintained by Vanessa Sochat, SuperKogito

watchme

A reproducible monitoring client that watches URLs, system resources, and more, with exporters for research tracking.

Maintained by Vanessa Sochat
WI

Witch Hazel

A color scheme inspired by magic and nature.

Maintained by Thea Flowers
